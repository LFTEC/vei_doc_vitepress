# 概述

为构建公开、公平、透明、可预期的供应链生态，实现供需双方高效协同、利益共享，万华化学物资装备部正式推行供应商生态库存VEI（Vendor Ecological Inventory） 对接模式。本手册面向所有有意向与万华采用 VEI 模式合作的供应商。本手册旨在完整、清晰的说明 VEI 对接模式、运行规则、分配逻辑、考核机制、冻结规则等。 

## 目标读者
阅读本手册前，您服务的公司需要是万华准入的供应商。

本手册包含 VEI 采购模式所有的基础概念、运营理念、对接方法、对接规范等。您不需要在阅读前了解其他的知识。  
本手册主要面向以下两种读者：
* **业务管理人员** 您需要了解您所在公司的销售与库存管理模式，以便能够在阅读后准确的评价贵方是否有意愿，且具备与万华进行 VEI 模式对接的能力
  
* **IT 技术人员** 您需要了解您所在公司的业务系统，并有能力评价系统是否具有与万华系统进行对接的能力。

## 接入概览
本手册包含两大部分的内容：**业务指南**与**技术指南**。业务指南主要向您介绍 VEI 模式的理念，收益及规则等知识。技术指南主要向您介绍对接的方法与思路，以及对接的流程，辅助您完成VEI 模式的对接。

## 版本管理
当前 VEI 系统的版本号为 v1.0。我们承诺在大版本不更新的前提下，所有的业务变更将向下兼容，以确保之前接入的供应商业务不受影响。

小版本的升级主要会包含如下内容：
* BUG 修复
* 小功能更新（不影响已接入的供应商，我们会确保原有接入功能的可用）
