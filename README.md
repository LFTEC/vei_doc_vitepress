## 接入总览
当供应商选择通过系统级的对接实现备库库存的对接，需按照如下的步骤进行。

## 接入申请
对于已经完成供应商准入流程的供应商，您此时已经拥有万华提供的供应商编号。如果已经与万华物资装备部协商好进行供应商生态库存的接入，请整理相关资料并提交给物装部采购员，并按照下面的流程进行供应商生态库存系统接入申请。

### 测试

## 完成开发
完成了生态库存接入申请的供应商，可以开始进行代码的开发工作。在进行开发之前，请确保您的系统开发语言符合下面的要求：

1. 您的系统需要具备接入 raabitMQ 消息队列的能力。您的开发语言必须为：
   * Java
   * C#
   * Python
   * Go
   * Javascript or Typescript
   * 其他开发语言（需要自行测试可接入性）
2. 您的系统能够通过 http api 实现系统对接能力。

## 联调测试
万华提供了一套完善的测试环境，提供双方系统的测试功能。需要完成所有相关的联调测试流程才能成为授权接入系统。具体的联调测试流程请见相关文档。

## 生产接入
仅当完成了所有的联调测试工作后，万华才会提供生产环境对接的账号、密码及密钥。将已经开发好的系统部署到生产环境中，并确保使用了正确的账号及密钥完成对接。
